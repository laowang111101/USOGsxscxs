# 个人健康管理系统 java456

## 项目概述

个人健康管理系统是一款集成了后端SpringBoot框架和MyBatis持久层框架，以及前端Vue.js框架的应用程序。该系统致力于为用户提供全面的个人健康管理服务。

## 技术栈

### 后端

- Spring Boot
- MyBatis

### 前端

- Vue.js

### 数据库

- MySQL 5.7

## 功能模块

系统主要包含以下功能模块：

1. 用户注册与登录
2. 个人健康档案管理
3. 健康数据统计分析
4. 健康资讯推送

## 系统角色

- 管理员：负责系统维护与内容管理
- 普通用户：注册后可管理个人健康信息

## 运行环境

- JDK 1.8 或更高版本
- MySQL 5.7
- 需要安装相应的IDE（如 IDEA 或 Eclipse）进行开发和运行

## 部署步骤

1. 准备开发环境：IDEA 或 Eclipse，JDK 1.8，MySQL 5.7
2. 导入项目代码到开发工具
3. 创建数据库，并导入SQL脚本
4. 配置项目中的数据库连接信息
5. 运行项目

## 目录结构

```
项目目录
├── src                    源代码目录
│   ├── main               主程序目录
│   │   ├── java           Java源文件目录
│   │   ├── resources      配置文件目录
│   │   └── webapp         前端资源目录
│   └── test               测试代码目录
└── pom.xml                Maven项目配置文件
```

## 核心亮点

- 前后端分离，易于维护和扩展
- 使用Spring Boot，提高开发效率
- 集成MyBatis，灵活处理数据库操作
